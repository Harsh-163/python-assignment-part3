# Integrated Python Project: Management & Analysis Systems

This repository contains a comprehensive three-part assignment demonstrating proficiency in Python basics, advanced data structures, and production-grade API integration with error handling.

---

## 📂 Project Overview

### 🏁 Part 1: Basics & Control Flow
* **Data Cleaning:** Normalizing student profiles and parsing mixed string/numeric data.
* **Logic:** Grading systems and dynamic input loops with `isdigit()` validation.

### 🍱 Part 2: Data Structures (Restaurant Management)
* **Nested Structures:** Managing a complex menu and inventory system using Dictionaries and Lists.
* **Memory Management:** Demonstrating `copy.deepcopy()` to maintain system backups.
* **Analysis:** Sales log aggregation and best-seller identification.

### 🌐 Part 3: File I/O & API Integration
* **File Operations:** Persistence through `python_notes.txt` (Write/Append/Read).
* **API Communication:** Interacting with the `DummyJSON` API using `GET` and `POST` methods.
* **Resilient Code:** Specialized `try-except` blocks for `ConnectionError` and `Timeout`.
* **Error Logging:** A timestamped logging utility (`error_log.txt`) that tracks both Python exceptions and HTTP failures (404s).

---

## 🛠️ Skills Summary
* **Languages:** Python 3.x
* **Libraries:** `requests`, `datetime`, `copy`, `json`
* **Techniques:** CRUD operations, API filtering/sorting, defensive programming, and tabular reporting.
